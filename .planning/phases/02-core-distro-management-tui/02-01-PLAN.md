3. `pub fn start_distro(&self, name: &str) -> Result<String, CoreError>` — runs `wsl.exe -d <name> -- true` (silent start without attaching)
4. `pub fn terminate_distro(&self, name: &str) -> Result<String, CoreError>` — runs `wsl.exe --terminate <name>`
5. `pub fn set_default(&self, name: &str) -> Result<String, CoreError>` — runs `wsl.exe --set-default <name>`
6. `pub fn unregister(&self, name: &str) -> Result<String, CoreError>` — runs `wsl.exe --unregister <name>`
7. `pub fn export_distro(&self, name: &str, path: &str) -> Result<String, CoreError>` — runs `wsl.exe --export <name> <path>`
8. `pub fn import_distro(&self, name: &str, install_dir: &str, tar_path: &str) -> Result<String, CoreError>` — runs `wsl.exe --import <name> <install_dir> <tar_path>`
9. `pub fn update_wsl(&self) -> Result<String, CoreError>` — runs `wsl.exe --update`

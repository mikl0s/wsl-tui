**Upgrade `wsl-tui/src/main.rs`:**

Replace the synchronous event loop with `EventStream` + `tokio::select!`:

```rust
use crossterm::event::{Event, EventStream, KeyCode, KeyEventKind};
use futures::StreamExt;
use tokio::time::{Duration, interval};

async fn run_app(
    terminal: &mut ratatui::DefaultTerminal,
    app: &mut App,
) -> anyhow::Result<()> {
    let mut events = EventStream::new();
    let mut poll_ticker = interval(Duration::from_secs(5));

    // Initial distro load
    if let Err(e) = app.refresh_distros() {
        // Log error but don't crash — distros will be empty
        eprintln!("Warning: failed to load distros: {e}");
    }

    while app.running {
        terminal.draw(|frame| ui::render(app, frame))?;

        tokio::select! {
            _ = poll_ticker.tick() => {
                let _ = app.refresh_distros();  // swallow errors during background refresh
            }
            maybe_event = events.next() => {
                let Some(Ok(event)) = maybe_event else { break };
                if let Event::Key(key) = event {
                    if key.kind != KeyEventKind::Press {
                        continue;
                    }
                    handle_key_event(app, &key);
                }
            }
        }
    }
    Ok(())
}
```

Implement `fn handle_key_event(app: &mut App, key: &KeyEvent)`:
- If `app.show_welcome`: dismiss on any key
- If `app.modal != ModalState::None`: route to modal-specific handler
- If `app.filter_active`: route char/backspace/esc to filter handlers
- Otherwise: map key to `Action` via `KeyBindings`, call `app.handle_action(action)` or execute the action in the event handler

Load `KeyBindings` in `main()` from the config: `let keybindings = KeyBindings::from_config(&config);`
